Proxy Parser

Outputs proxy values into readable format

Description:
Takes in full proxy string, finds the bits assosiated with each category, and outputs it into a text file as a list

Libraries:
Imports os library to read proxy string and write text into an output file accordingly

Executing Program:
Paste your proxy string into "proxystring.txt"
Run script
Find your final output with category types listed in "proxyoutput.txt"
